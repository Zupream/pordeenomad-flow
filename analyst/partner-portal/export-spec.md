# Export Specification — PordeeConnect Partner Portal

รายละเอียดไฟล์ที่ Export จากแต่ละหน้าใน Partner Portal
ทุกไฟล์เป็น `.xlsx` (Excel) ยกเว้นระบุเป็นอย่างอื่น

---

## 1. สรุปรายได้

- **ชื่อไฟล์:** `สรุปรายได้_{ช่วงวันที่}.xlsx`
- **ตัวอย่าง:** `สรุปรายได้_พ.ค._2569.xlsx`, `สรุปรายได้_1พ.ค.-15พ.ค._2569.xlsx`
- **ข้อมูลตาม:** filter วันที่ที่เลือกอยู่
- **Export จากหน้า:** รายได้ของฉัน (ตารางสรุปรายได้)
- **แหล่งข้อมูล:** รวมจากทุกประเภทรายได้

| คอลัมน์ | ตัวอย่างข้อมูล | ที่มา |
|---------|---------------|-------|
| ประเภทรายได้ | ค่าคอมมิชชั่น Affiliate | ตารางสรุปรายได้ |
| จำนวน (฿) | 8,200 | คำนวณจากออเดอร์/ลูกทีมตามช่วงเวลา |


**แถวสุดท้าย:** รวมทั้งหมด + จำนวนรวม

## 3. Royalty / Licensing Fee (รายลูกทีม)

- **ชื่อไฟล์:** `Royalty_{ช่วงวันที่}.xlsx`
- **ตัวอย่าง:** `Royalty_พ.ค._2569.xlsx`
- **ข้อมูลตาม:** filter วันที่ที่เลือกอยู่
- **Export จากหน้า:** รายละเอียด Royalty / Licensing Fee
- **แหล่งข้อมูล:** ตาราง Team Members + Orders + Royalty Pool calculation

| คอลัมน์ | ตัวอย่างข้อมูล | ที่มา |
|---------|---------------|-------|
| ชื่อลูกทีม | วรรณา สุขใจ | members.name |
| รหัสสมาชิก | #PN-012 | members.code |
| ชั้น | 1 | members.level (สัมพัทธ์กับเรา) |
| ตำแหน่ง | Gold | members.tier |
| ยอดขายลูกทีม (฿) | 12,400 | sum(orders) ของลูกทีมคนนั้นในช่วงเวลา |
| Royalty Pool 2% (฿) | 248 | ยอดขาย × 2% |
| ค่าลิขสิทธิ์ที่ได้ (฿) | 74.40 | คำนวณจาก cascade ตาม tier |

**แถวสุดท้าย:** รวม + จำนวนค่าลิขสิทธิ์ทั้งหมด

## 6. ประวัติการถอนเงิน

- **ชื่อไฟล์:** `ประวัติการถอนเงิน_{วันที่ export}.xlsx`
- **ตัวอย่าง:** `ประวัติการถอนเงิน_10พ.ค.2569.xlsx`
- **ข้อมูลตาม:** filter สถานะที่เลือกอยู่ (ทั้งหมด / รอดำเนินการ / สำเร็จ / ปฏิเสธ)

## 7. ลูกทีมทั้งหมด

- **ชื่อไฟล์:** `ลูกทีมทั้งหมด_{วันที่ export}.xlsx`
- **ตัวอย่าง:** `ลูกทีมทั้งหมด_10พ.ค.2569.xlsx`
- **ข้อมูลตาม:** **ไม่ขึ้นกับ filter** — export ลูกทีมทั้งหมดทุกชั้นเสมอ
- **Export จากหน้า:** ทีมของฉัน
- **แหล่งข้อมูล:** ตาราง Team Members (เฉพาะลูกทีมของ user ชั้น 1-3)
- **ข้อมูลที่ไม่รวม (privacy):** ไม่แสดงว่าลูกทีมมีลูกทีมเป็นใคร / ไม่แสดงคอลัมน์ "แม่ทีม"

| คอลัมน์ | ตัวอย่างข้อมูล | ที่มา |
|---------|---------------|-------|
| ชื่อ | วรรณา สุขใจ | members.name |
| รหัสสมาชิก | #PN-012 | members.code |
| ตำแหน่ง | Gold | members.tier |
| ชั้น | 1 | members.level (สัมพัทธ์กับเรา) |
| ยอดขาย (฿) | 12,400 | sum(orders) ล่าสุด |
| ค่าคอม Affiliate (฿) | 3,200 | commission income ล่าสุด |

**ไม่มีคอลัมน์:** แม่ทีม, จำนวนลูกทีม (เพื่อ privacy ไม่เปิดเผยความสัมพันธ์ระหว่างลูกทีมกันเอง)

**แถวสุดท้าย:** สรุปจำนวนลูกทีมแยกตามชั้น (ชั้น 1: X คน, ชั้น 2: X คน, ชั้น 3: X คน, รวม: X คน)

## หมายเหตุทั่วไป

- ทุกไฟล์ยกเว้น "ลูกทีมทั้งหมด" จะ export ตาม filter ที่ user เลือกอยู่ ณ ขณะนั้น
- ชื่อไฟล์ต่อท้ายด้วยช่วงเวลาที่ filter อยู่ เพื่อให้ user จัดการไฟล์ได้ง่าย
- ถ้า user เลือก filter แบบ custom date range ให้แสดงเป็น `{วันเริ่ม}-{วันสิ้นสุด}` เช่น `1พ.ค.-15พ.ค._2569`
- ทุกไฟล์มี header row (แถวหัวคอลัมน์) และ summary row (แถวรวม) ท้ายตาราง
- จำนวนเงินแสดงเป็น number format 2 ทศนิยม (เช่น 8,200.00)
- วันที่แสดงในรูปแบบ `DD/MM/YYYY HH:MM` (เช่น 05/05/2569 14:32)
