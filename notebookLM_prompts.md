## Prompt 1: สร้างตัวอย่างครบทุกเคส (แนะนำใช้อันนี้เป็นอันแรก)

```
จากเอกสารโครงสร้าง Commission ของ Pordeenomad ช่วยยกตัวอย่างการคำนวณแบบ step-by-step ให้ครบทุกเคสที่เป็นไปได้ โดยใช้สินค้าราคา 1,000 บาท Commission 20% Platform Fee 3% เป็นฐาน

เคสที่ต้องการ:
1. Affiliate ไม่มีแม่ทีม (สมัครเอง) — ทุก Tier ของ Affiliate (Customer, Silver, Gold, Platinum, Wisdom)
2. Affiliate มีแม่ทีม — แม่ทีมเป็นแต่ละ Tier (Silver, Gold, Platinum, Wisdom)
3. กรณี Cross-Tier — แม่ทีมเป็น Silver แต่ลูกทีมเป็น Gold / แม่ทีมเป็น Platinum แต่ลูกทีมเป็น Customer
4. กรณีลูกค้าซื้อซ้ำ (Lifetime Binding) — Affiliate ได้เท่าไหร่ทุกครั้ง
5. กรณีมี Order คืนสินค้า — เกิดอะไรขึ้นกับ Commission

แต่ละเคส ให้แสดง:
- สถานการณ์ (ใครเป็นใคร Tier อะไร)
- การคำนวณทีละขั้น (Pool → ก้อน 1 → ก้อน 2 → ใครได้เท่าไหร่)
- ตารางสรุปว่าแต่ละคนได้เท่าไหร่
- Seller เหลือเท่าไหร่

ใช้ภาษาไทย อธิบายให้คนที่ไม่มีพื้นฐานเข้าใจได้
```

## Prompt 2: อธิบายระบบให้คนทั่วไปเข้าใจ

```
ช่วยอธิบายระบบ Commission ของ Pordeenomad ให้เข้าใจง่ายๆ เหมือนกำลังอธิบายให้เพื่อนที่ไม่มีพื้นฐานเลยฟัง

ครอบคลุมหัวข้อนี้:
1. ระบบทำงานยังไง (เป็นภาษาชาวบ้าน)
2. รายได้ 5 ประเภทคืออะไร พร้อมตัวอย่างง่ายๆ แต่ละอัน
3. ระดับสมาชิก (Tier) ต้องทำอะไรบ้างถึงจะอัพเกรด
4. ถ้าเป็น Affiliate ใหม่ จะเริ่มต้นยังไง ได้อะไรบ้าง
5. ถ้าเป็น Seller ต้องจ่ายเท่าไหร่ เหลือเท่าไหร่

ใช้ภาษาง่ายๆ ไม่ใช้ศัพท์เทคนิค หรือถ้าจำเป็นให้อธิบายกำกับ
ยกตัวอย่างเป็นตัวเลขจริงทุกข้อ
```

## Prompt 3: เปรียบเทียบ Pordeenomad vs BStore

```
จากเอกสาร ช่วยเปรียบเทียบ Pordeenomad กับ BStore แบบละเอียด

สิ่งที่ต้องการ:
1. ตารางเปรียบเทียบแต่ละหัวข้อ (โมเดลธุรกิจ, ใครจ่ายอะไร, โครงสร้าง Commission)
2. ยกตัวอย่างตัวเลขเดียวกัน (สินค้า 1,000 บาท) ทั้ง 2 ระบบ แสดงให้เห็นว่าเงินไหลต่างกันยังไง
3. ข้อดี-ข้อเสียของแต่ละโมเดล
4. ทำไม Pordeenomad ถึงใช้แบบ BStore ตรงๆ ไม่ได้
5. สิ่งที่ Pordeenomad อ้างอิงจาก BStore vs สิ่งที่ปรับเปลี่ยน
```

## Prompt 4: สร้าง FAQ สำหรับ Affiliate

```
จากเอกสาร ช่วยสร้าง FAQ (คำถามที่พบบ่อย) สำหรับ Affiliate ที่จะใช้ระบบ Pordeenomad

หัวข้อที่ควรมี:
- สมัครยังไง ต้องจ่ายเงินไหม
- รายได้มีกี่ประเภท ได้ยังไงบ้าง
- Lifetime Binding คืออะไร
- อัพเกรด Tier ต้องทำอะไร
- ค่าลิขสิทธิ์แม่ทีมคืออะไร (ทำไมเงินถูกหัก)
- ถอนเงินยังไง
- ถ้าลูกค้าคืนสินค้า Commission เป็นยังไง
- ต่างจาก Affiliate ทั่วไปยังไง

ตอบแต่ละข้อให้สั้น กระชับ เข้าใจง่าย ใช้ตัวเลขตัวอย่างประกอบ
```

## Prompt 5: สร้าง FAQ สำหรับ Seller

```
จากเอกสาร ช่วยสร้าง FAQ สำหรับ Seller (ร้านค้า) ที่จะลงขายบน Pordeenomad

หัวข้อที่ควรมี:
- ค่าใช้จ่ายทั้งหมดเท่าไหร่ (Commission + Platform Fee)
- Commission ที่ตั้งไว้ถูกแบ่งยังไง ใครได้อะไร
- ตั้ง Commission เท่าไหร่ดี
- ถ้าตั้ง Commission 10% vs 20% ต่างกันยังไง
- เงินจะเข้าบัญชี Seller เมื่อไหร่
- ถ้ามีการคืนสินค้าเป็นยังไง
- Platform Fee คิดยังไง

ตอบแต่ละข้อพร้อมตัวอย่างตัวเลข เช่น สินค้าราคา 500 / 1,000 / 3,000 บาท
```

## Prompt 6: วิเคราะห์จุดเสี่ยง & ข้อควรระวัง

```
จากเอกสารโครงสร้าง Commission ของ Pordeenomad ช่วยวิเคราะห์:

1. จุดเสี่ยงที่อาจเกิดปัญหา (เช่น กรณีที่ Commission ไม่พอจ่าย, Affiliate ไม่พอใจ)
2. กรณี Edge Case ที่อาจทำให้ระบบมีปัญหา
3. ข้อควรระวังเรื่อง Lifetime Binding
4. ความเสี่ยงเรื่อง Reserve Policy (เงินก้อน 2)
5. สิ่งที่ยังต้อง Confirm — และถ้าตัดสินใจผิดจะมีผลกระทบอะไร

ยกตัวอย่างสถานการณ์จริงที่เป็นไปได้ แต่ละข้อ
```

# วิธีใช้

1. เปิด [NotebookLM](https://notebooklm.google.com/)
2. สร้าง Notebook ใหม่
3. อัพโหลดไฟล์ `notebookLM_source.md` เป็น Source
4. Copy prompt ที่ต้องการจากด้านบน แล้ว paste ลงในช่อง chat
5. NotebookLM จะตอบโดยอ้างอิงจากเอกสารที่อัพโหลด

**Tips:**
- ใช้ Prompt 1 ก่อน เพื่อให้ได้ตัวอย่างครบทุกเคส
- ถ้าต้องการ Audio Podcast ใช้ฟีเจอร์ "Audio Overview" ของ NotebookLM โดยตรง (ไม่ต้องใช้ prompt — แค่กดปุ่ม Generate)
- สามารถถามต่อยอดได้ เช่น "ช่วยเพิ่มเคสที่สินค้าราคา 5,000 บาท" หรือ "ถ้า Commission เป็น 10% แทน 20% จะเป็นยังไง"
